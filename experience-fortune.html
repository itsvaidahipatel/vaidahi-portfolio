Software Engineer Co-op

Fortune Panel Systems
January 2022 – December 2022

Overview

Worked as a Software Engineer Co-op on an IoT-based data processing system for road construction plants. Focused on building real-time monitoring dashboards, data collection systems, and optimizing visualization performance.

Key Achievements

  • Aided in creating IoT-based data processing system using TLS-secured WebSockets to collect and stream telemetry from 1,000+ road construction plants
  • Collected aggregate TPH, bitumen %, filler %, GPS coordinates, and other machine telemetry data
  • Built and integrated dynamic Razor Pages dashboards and data-entry forms with Excel export and search functionality
  • Enabled operators, engineers, and customers to monitor machine performance, production metrics, and location data in real time
  • Optimized ASP.NET Core Razor Pages dashboards with server-side rendering and on-demand data fetching
  • Enabled seamless real-time visualization of machine telemetry for operators and clients

Technical Contributions

  • Created IoT-based data processing system connecting 1,000+ road construction plants
  • Implemented TLS-secured WebSockets for secure telemetry collection
  • Streamed telemetry data (aggregate TPH, bitumen %, filler %, GPS coordinates) to centralized PostgreSQL database
  • Built dynamic Razor Pages dashboards with real-time visualization
  • Created data-entry forms with Excel export and search functionality
  • Optimized dashboards with server-side rendering for better performance
  • Implemented on-demand data fetching via AJAX/SignalR for real-time updates
  • Enabled multi-user access for operators, engineers, and customers
Impact: Created a comprehensive IoT monitoring system connecting 1,000+ devices, enabling real-time monitoring of road construction operations. Improved operational visibility and decision-making through dynamic dashboards and data visualization.
